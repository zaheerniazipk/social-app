### How to Run a Django Project

1. Install pipenv using cmd
```
-> pip install pipenv
```

2. Open and activate pipenv virtual environment in the project folder
```
-> pipenv
-> pipenv shell
```

3. Run Django Project
```
-> python manage.py runserver
```
